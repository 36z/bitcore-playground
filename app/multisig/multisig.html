This is how it works

“2014 is the year of multisig” said Gavin Andresen at Bitcoin2014 in Amsterdam. Bitcore provides an easy to use interface to create an address for which funds can only be redeemed if signed by the owners of the private keys corresponding to a subset of public keys. This interactive example allows you to create an address from a set of public or private keys.

Read more in the bitcore developer guide.
